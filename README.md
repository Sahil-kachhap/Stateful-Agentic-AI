# Stateful-Agentic-AI

## Getting Started
1. Fork this Repo
2. Clone your Forked Repo: `git clone forked-repo-url`
3. Create a branch out of your main branch - `git checkout -b "your-feature-name"`
4. Setup your codebase by installing necessary dependencies
5. Once you have completed your code changes, commit the changes and push the changes to remote: `git push origin main`
6. Create a Pull request from your repo to this repo to merge your changes.
